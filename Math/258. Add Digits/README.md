Time spent : 1 min (see hint)

Solution : Digit root

GIVEN: a non-negative integer num

RETURNS: only one digit, repeately add all its digits

EXAMPLES:

Input: 

```
38
```

Output: 2 (3 + 8 = 11, 1 + 1 = 2)
