Time spent :  5 min(self)

GIVEN: a string	

RETURNS: the same string in lowercase

EXAMPLES:

**Example 1:**

```
Input: "Hello"
Output: "hello"
```

**Example 2:**

```
Input: "here"
Output: "here"
```

**Example 3:**

```
Input: "LOVELY"
Output: "lovely"
```

Algorithm:

for each character change the char to lowercase, then combine them together.
